# txt2dsl
A simple interactive python script that convert txt (tab) dictionary that has converted from pyglossary to dsl dictionary.

It is the first trial version.

How to use it?

- Convert any dictionary you prefer using pyglossary to .txt extension and come to here to convert that to .dsl Lingvo dictionary.

- Download this python file and put it beside the dictionary file and run:
     python txt2dsl.py
  then enter the input file name with .txt extension and wait until finished, now you have your .dsl dictionary. Enjoy

- Note: This script is made only for my using, I just decided to make it public for those who are in need.
